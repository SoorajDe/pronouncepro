## 🚀 Overview

**PronouncePro** is a modern, full-stack web application designed to help users master pronunciation across multiple languages using cutting-edge AI analysis. Built with a responsive glassmorphism UI, it provides an immersive learning experience with real-time speech recognition, AI-powered feedback via Google's Gemini model, and comprehensive progress tracking.

### 🎯 Key Features

- **Multi-Language Support**: English, Spanish, French, Hindi, Kannada, Tamil, Telugu
- **AI-Powered Analysis**: Real-time pronunciation scoring using Gemini AI  
- **Speech Recognition & Synthesis**: Browser-native Web Speech API
- **Progressive Web App (PWA)**: Installable, offline-capable
- **Complete Authentication**: Register, login, forgot password, JWT tokens
- **10+ Learning Modules**: Practice, Roleplay, Games, Exams, Scanner, Analytics
- **Streak Tracking**: Daily practice streaks with persistence
- **Dark/Light Mode**: Automatic + manual theme toggle
- **Mobile-First Design**: Perfect responsive experience

## 🛠 Tech Stack

**Backend**: Node.js + Express + lowdb + JWT + bcrypt + Gemini API + Nodemailer  
**Frontend**: Vite + Vanilla JS + Tailwind + Chart.js + Tesseract.js + PWA

## 🚀 Quick Start

1. `cd backend && npm install && npm start`
2. `cd frontend && npm install && npm run dev`
3. Add `GEMINI_API_KEY` to `backend/.env`
4. Visit `http://localhost:5173`
